Escopo inicial:
	
	Contas a Pagar
    
    Criar um sistema que ajude a gerenciar minhas financas, 
    gerando planejamentos de ciclo mensal e anual de contas a pagar.

	=> Devera haver uma interface onde sera possivel visualizar as contas a pagar de 
		um ciclo que sera mensal e/ou anual

    => A interface devera ter um botao onde o usuario podera modificar o status de uma conta a pagar

    => Uma Conta a Pagar tera os seguintes status: Pendente, Paga, Atrasada e Paga Atrasada

    => Devara haver uma interface onde sera possivel cadastrar uma nova Conta

    => Devara haver uma interface onde sera possivel editar uma Conta

    => Devera gerar automaticamente o planejamento financeiro mensal, 
	a partir das contas cadastradas e compras efetuadas;
